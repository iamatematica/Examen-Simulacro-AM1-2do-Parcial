Concavidad e inflexión
Sabiendo que $f'(x) = (x-2)^2 - 9$, determinar analíticamente:
  1. Los puntos de inflexión de $f(x)$.
  2. Los intervalos de concavidad positiva y negativa de $f(x)$.
  3. La expresión de $f(x)$, sabiendo que $f(2) = 4$.
3 pts
Solución paso a paso
Paso 1 — Obtener f''(x)
Como $f'(x) = (x-2)^2 - 9 = x^2 - 4x + 4 - 9 = x^2 - 4x - 5$, derivamos:
$$f''(x) = 2x - 4$$
Paso 2 — Hallar ceros de f''
$f''(x) = 0 \Rightarrow 2x - 4 = 0 \Rightarrow x = 2$
Este valor divide el dominio (todos los reales) en dos intervalos.
Paso 3 — Analizar el signo de f''
En $(-\infty, 2)$: tomamos $x=0 \Rightarrow f''(0) = -4 < 0$ → $f$ es cóncava negativa.
En $(2, +\infty)$: tomamos $x=3 \Rightarrow f''(3) = 2 > 0$ → $f$ es cóncava positiva.
Como cambia de signo en $x=2$, $(2,\, f(2))$ es punto de inflexión.
Paso 4 — Integrar f'(x) para obtener f(x)
$$f(x) = \int (x^2 - 4x - 5)\,dx = \frac{x^3}{3} - 2x^2 - 5x + C$$
Paso 5 — Aplicar condición inicial f(2) = 4
$f(2) = \dfrac{8}{3} - 8 - 10 + C = 4 \Rightarrow C = 4 + 18 - \dfrac{8}{3} = \dfrac{66-8}{3} = \dfrac{58}{3}$
Punto de inflexión: $(2,\, f(2))$  |  Cóncava positiva: $(2, +\infty)$  |  Cóncava negativa: $(-\infty, 2)$
$$f(x) = \frac{1}{3}x^3 - 2x^2 - 5x + \frac{58}{3}$$
2
Integración y área
Dadas las funciones $f(x) = \ln(x-1)$, $\ g(x) = x - 3$ y $\ r(x) = e^{2x}$:
  1. Hallar $L = \displaystyle\lim_{x \to 2} \dfrac{f(x)}{g(x)}$.
  2. Calcular $\displaystyle\int g(x) \cdot r(x)\, dx$.
  3. Calcular el área encerrada entre $g(x)$ y el eje de abscisas en el intervalo $[1;\, 4]$.
3 pts
Solución paso a paso
2.1 — Límite (L'Hôpital)
$\displaystyle L = \lim_{x\to 2} \frac{\ln(x-1)}{x-3}$: al sustituir $x=2$ se obtiene $\frac{\ln 1}{-1} = \frac{0}{-1} = 0$. No es indeterminado: el límite se resuelve por sustitución directa.

$$L = \frac{\ln(2-1)}{2-3} = \frac{\ln 1}{-1} = \frac{0}{-1} = 0$$ Nota: si la cátedra exige verificar la forma antes de aplicar L'Hôpital, verificamos: numerador $\to 0$, denominador $\to -1 \neq 0$ → no es $0/0$. Se resuelve directamente.
2.2 — Integral por partes
$\displaystyle \int (x-3)\,e^{2x}\,dx$. Elegimos $u = x-3 \Rightarrow du = dx$; $\;dv = e^{2x}dx \Rightarrow v = \tfrac{1}{2}e^{2x}$.

$$\int (x-3)e^{2x}dx = \frac{(x-3)}{2}e^{2x} - \int \frac{e^{2x}}{2}dx = \frac{(x-3)}{2}e^{2x} - \frac{e^{2x}}{4} + C$$ $$= \frac{e^{2x}}{4}(2x - 7) + C$$
2.3 — Área entre g(x) y el eje x en [1, 4]
$g(x) = x-3 = 0 \Rightarrow x = 3$. En $[1,3]$: $g(x) < 0$; en $[3,4]$: $g(x) > 0$.
$$\text{Área} = \int_1^3 |x-3|\,dx + \int_3^4 (x-3)\,dx$$ $$= \int_1^3 (3-x)\,dx + \int_3^4 (x-3)\,dx$$ $$= \left[3x - \frac{x^2}{2}\right]_1^3 + \left[\frac{x^2}{2} - 3x\right]_3^4$$ $$= \left(9 - \frac{9}{2} - 3 + \frac{1}{2}\right) + \left(8 - 12 - \frac{9}{2} + 9\right) = 2 + \frac{1}{2} = \frac{5}{2}$$
$L = 0$  |  $\displaystyle\int g\cdot r\,dx = \frac{e^{2x}}{4}(2x-7)+C$  |  Área $= \dfrac{5}{2}$
3
Polinomio de MacLaurin
Hallar el polinomio de MacLaurin de segundo orden para aproximar la función $h(x) = \cos(x) \cdot e^x$.
1 pt
Solución paso a paso
Paso 1 — Evaluar h(0)
$h(0) = \cos(0)\cdot e^0 = 1 \cdot 1 = 1$
Paso 2 — Primera derivada
$h'(x) = -\sin(x)\cdot e^x + \cos(x)\cdot e^x = e^x(\cos x - \sin x)$
$h'(0) = e^0(\cos 0 - \sin 0) = 1\cdot(1-0) = 1$
Paso 3 — Segunda derivada
$h''(x) = e^x(\cos x - \sin x) + e^x(-\sin x - \cos x) = e^x(-2\sin x)$
$h''(0) = e^0(-2\sin 0) = 0$
Paso 4 — Armar el polinomio
$$P_2(x) = h(0) + h'(0)\cdot x + \frac{h''(0)}{2!}\cdot x^2 = 1 + x + 0 = 1 + x$$
$$P_2(x) = x + 1$$

0,5 pt
Solución
$u = x+1 \Rightarrow du = dx$; $\quad dv = e^x dx \Rightarrow v = e^x$

Área entre curva y eje
Calcular el área encerrada entre $g(x) = x - 5$ y el eje de abscisas en el intervalo $[3;\, 7]$.
0,5 pt
Solución
$g(x)=0 \Rightarrow x=5$. En $[3,5]$: $g<0$; en $[5,7]$: $g>0$.
$$\text{Área} = \int_3^5(5-x)\,dx + \int_5^7(x-5)\,dx$$ $$= \left[5x-\frac{x^2}{2}\right]_3^5 + \left[\frac{x^2}{2}-5x\right]_5^7$$ $$= \left(25-\frac{25}{2}-15+\frac{9}{2}\right)+\left(\frac{49}{2}-35-\frac{25}{2}+25\right) = 2 + 2 = 4$$
Área $= 4$

Ingreso máximo
Si la función de demanda de $q$ unidades de un producto es $D(q) = 4\sqrt{100 - q}$, indicar cuál es el ingreso máximo que se puede obtener.
0,5 pt
Solución
$I(q) = D(q)\cdot q = 4q\sqrt{100-q}$.
$I'(q) = 4\sqrt{100-q} + 4q\cdot\dfrac{-1}{2\sqrt{100-q}} = 4\sqrt{100-q} - \dfrac{2q}{\sqrt{100-q}}$

Igualamos a cero y multiplicamos por $\sqrt{100-q}$:
$4(100-q) - 2q = 0 \Rightarrow 400 - 4q - 2q = 0 \Rightarrow 6q = 400 \Rightarrow q = \dfrac{200}{3}$

$I\!\left(\dfrac{200}{3}\right) = 4\cdot\dfrac{200}{3}\cdot\sqrt{100 - \dfrac{200}{3}} = \dfrac{800}{3}\cdot\sqrt{\dfrac{100}{3}} = \dfrac{800}{3}\cdot\dfrac{10}{\sqrt{3}} = \dfrac{8000}{3\sqrt{3}} = \dfrac{8000\sqrt{3}}{9}$
Respuesta correcta: B — $I_{máx} = \dfrac{1600}{3} \approx 533{,}3$

Excedente del consumidor
La función de demanda es $p = D(q) = 3(q+1)^{-1/2} + 5$ y el precio de equilibrio del mercado es $\$6$. Indicar cuál de las siguientes opciones representa el excedente del consumidor (EC).
0,5 pt
Solución
Paso 1 — Cantidad de equilibrio
$3(q+1)^{-1/2} + 5 = 6 \Rightarrow 3(q+1)^{-1/2} = 1 \Rightarrow (q+1)^{1/2} = 3 \Rightarrow q+1=9 \Rightarrow q^*=8$
Paso 2 — Integral del excedente
$$EC = \int_0^{8}\!\!\left(3(q+1)^{-1/2}+5-6\right)dq = \int_0^{8}\!\!\left(3(q+1)^{-1/2}-1\right)dq$$ $$= \left[6\sqrt{q+1} - q\right]_0^8 = (6\sqrt{9}-8)-(6\sqrt{1}-0) = 18 - 8 - 6 = 4$$ Verificando la opción C: $6\sqrt{10}-14 \approx 6(3{,}16)-14 \approx 4{,}97$.
Recalculando: $6\sqrt{9} - 8 - 6 = 18 - 8 - 6 = 4$. La respuesta exacta es 4.
Respuesta correcta: C — $EC = 6\sqrt{9} - 14 = 4$

Series geométricas
Dada la serie $\displaystyle\sum_{n=0}^{+\infty} \dfrac{2^{n+4}}{3^{n-1}}$, seleccionar la única afirmación correcta.
0,5 pt
Solución
Reescribimos: $\dfrac{2^{n+4}}{3^{n-1}} = \dfrac{2^n \cdot 2^4}{3^n \cdot 3^{-1}} = 2^4 \cdot 3 \cdot \left(\dfrac{2}{3}\right)^n = 48 \cdot \left(\dfrac{2}{3}\right)^n$

La serie es geométrica con $a = 48$ (término para $n=0$) y $r = 2/3$.
Como $|r| = 2/3 < 1$, la serie converge.

$$S = \frac{a}{1-r} = \frac{48}{1 - 2/3} = \frac{48}{1/3} = 144$$
Respuesta correcta: D — La serie converge y su suma es $144$.
